MySQL常用指令：
mysql服务的启动和停止

连接数据库：mysql -u root -p
退出连接：exit
设置用户名密码：mysqladmin -u 用户名（随便起开心就好） -password 密码（你要设定的）
修改密码：mysqladmin -u 用户名 -p password 密码（你想修改成的）
创建库：create database 数据库的名字；（后边有个分号）
切换数据库：use 数据库名；
删除库：drop database 数据库名字；（后边有个分号）
查看数据库：show databases；（后边有个分号）
显示表：show tables；（后边有个分号）
新建表：create table 表的名字 （字段1 类型 （最多多少字符），以此类推你想写多少都行）；（后边有个分号）
